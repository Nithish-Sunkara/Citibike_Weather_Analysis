# CitiBike & NYC Weather Data Analysis

## Project Overview
This project analyzes CitiBike trip data and integrates it with NYC weather data to identify usage patterns, weather impact, and station-level dynamics.

## Tools Used
- MySQL
- Power BI
- GitHub

## Workflow
1. Data Ingestion
2. Data Cleaning & Staging
3. Data Validation
4. Feature Engineering
5. Data Integration
6. KPI Development
7. Dashboard Creation

## Dataset Sources
- CitiBike Trip Data (2019 Sample)
- NOAA NYC Weather Data
